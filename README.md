Install truffle for Dune/Tezos:
```
npm i -g @dune-network/truffle
```

Install dependencies:
```
npm install
```

Compile contracts:
```
npm run compile
```


Start a local node:
```
(ix create sandbox)
ix start
```

Deploy contracts:
```
npm run migrate
```

(for now, this is failing. We will debug it soon !)

Install dependencies in client/:
```
cd client
npm install
```

Build the client:
```
npm run build
```

Start a local server:
```
npm run start
```
